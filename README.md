# Kort Website

![Kort Logo](img/logo.svg)

Официальный сайт коммерческой организации **Kort**.

## О проекте

Статический сайт компании Kort с разделами:

- Главная
- О компании
- Контакты
- Аренда помещений
- Публичные пространства
- Партнёры
- Оплата
- Личный кабинет
- Панель модератора
- Вход и восстановление пароля

## Технологии

- HTML5
- CSS3
- JavaScript

## Структура проекта

/
├── img/                # Изображения
├── js/                 # JavaScript-файлы
├── Plan Zdaniy/        # Планы зданий
├── styles.css          # Основные стили
├── kort.html           # Главная страница
└── *.html              # Остальные страницы

## Запуск

Откройте `kort.html` в браузере или разверните через GitHub Pages / любой статический хостинг.
