* Логотип настоящей компании представлен в образовательных целях и он не имеет ничего общего с настоящим лого.

Дипломная работа, вебсайт для коммерческой организации **Kort**.

## О проекте

Статический сайт компании Kort с разделами:

- Главная
- О компании
- Контакты
- Аренда помещений
- Публичные пространства
- Партнёры
- Оплата
- Личный кабинет
- Панель модератора
- Вход и восстановление пароля

## Стек

- HTML5
- CSS3
- JavaScript
